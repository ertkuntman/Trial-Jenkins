Whenever you open the mashup it will be filtered with he latest [Fiscal Month-Year] available and with [Material Taxonomy]=Relevant. In order to do that we have to look at the Triggers.js that is inside "services". The structure of the trigger is arleady done. A trigger is a function that will be called when the application is opened and is useful to make filters by default. Here we create a function that is going to look at the latest fiscal month with the "createGenericObject" and "selectMatch" methods. For filtering [Material Taxonomy], we only need to use "selectMatch" method.
